AST节点TemplateDef

TemplateDef节点表示模板定义语句,包含一个Identifier节点,一个Parameter节点的列表,一个Identifier节点,一个Member节点的列表.

备注
这是一个具体的节点,其包括有子节点.
TemplateDef : 'template' Identifier '[' Parameter* ']' 'is' Identifier '(' Member* ')' ;
