Detailed Description

AST node Expression.

The Expression node represents an expression, which contains a Literal node, an Identifier node, and an Operation node.

Remarks
This is an abstract node that does not include child nodes.
Expression : Literal
| Identifier
| Operation
| ObjectRef
| MapRef
| TemplateRef
| ObjectIns
| TemplateParam
