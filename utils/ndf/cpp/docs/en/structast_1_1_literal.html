Detailed Description

AST node Literal.

The Literal node represents a literal, which contains a GUID node, a Path node, a Pair node, a Boolean node, a String node, an Integer node, a Float node, a Nil node, and a Vector node.

Remarks
This is an abstract node that does not include child nodes.
