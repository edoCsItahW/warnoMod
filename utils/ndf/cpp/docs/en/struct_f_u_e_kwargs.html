Detailed Description

The keyword arguments of function Parser::fromUntilExpect()

It is used to control the behavior of function Parser::fromUntilExpect(). It includes: firstStop, skipNewLine, debug

◆ firstStop

bool FUEKwargs::firstStop = false

Whether to stop when the first token type is not the expected type.

When firstStop is true, if the first token type is not the expected type, the function Parser::fromUntilExpect() will immediately return false and will not continue to traverse the token list.

See also
Parser::fromUntilExpect()

◆ skipNewLine

bool FUEKwargs::skipNewLine = true

Whether to skip the newline character.

When skipNewLine is true, if the function Parser::fromUntilExpect() encounters a newline character, it will skip it.

Note
Note that if the expected token type is TokenType::NEWLINE, even if skipNewLine is true , it will not skip the newline character.
